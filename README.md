# SubATE-WPF
Вариант 9: Абоненты АТС

## Требования

### Основные
- [ ] Должна содержать минимум 2 формы (основная с таблицей, без возможности редактирования данных и отдельное окно редактирования)
- [ ] Информация должна храниться в файлах.
- [ ] Программа должна позволять добавлять новую запись, редактировать и удалять через формы.
- [ ] Сущность, содержащая информацию, должна быть оформлена как класс и должна иметь 5 свойств разного типа.
- [ ] Для связи визуальных полей и данных в программе необходимо использовать механизм DataBinding.
- [ ] Программа должна содержать минимум 10 типов элементов управления.
- [ ] Реализовать обработку ошибок при вводе данных.
- [ ] В программе должно быть меню и контекстное меню.

### Дополнительные
- [ ] В случае большого количества данных в файлах, загружать данные из файла необходимо по частям.

## Элементы управления

1. Label
2. TextBox
3. CheckBox
4. ComboBox
5. DatePicker
6. Button
7. Menu
8. ContextMenu
9. DataGrid
10. Image
