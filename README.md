## Требования

### Основные
- [x] Должна содержать минимум 2 формы (основная с таблицей, без возможности редактирования данных и отдельное окно редактирования)
- [x] Информация должна храниться в файлах.
- [x] Программа должна позволять добавлять новую запись, редактировать и удалять через формы.
- [x] Сущность, содержащая информацию, должна быть оформлена как класс и должна иметь 5 свойств разного типа.
- [x] Для связи визуальных полей и данных в программе необходимо использовать механизм DataBinding.
- [x] Программа должна содержать минимум 10 типов элементов управления.
- [ ] Реализовать обработку ошибок при вводе данных.
- [x] В программе должно быть меню и контекстное меню.

### Дополнительные
- [x] В случае большого количества данных в файлах, загружать данные из файла необходимо по частям.
